Program Description:
This project is called Weather Dashboard with Advice. It connects to a live weather API to get weather data for a city entered by the user. The program gives the current weather, also clothing advice, and gives travel advice based on the weather condition. It also saves each search in a history file, allows the user to view previous searches, search for a city in the history, sort the history by temperature, and display a forecast trend chart using matplotlib. The program includes error handling for invalid input, missing files, wrong city names, and network problems.

Libraries Used:
- requests
- matplotlib

Installation Instructions:
1. Install Python.
2. Open terminal in the project folder.
3. Run: pip install -r requirements.txt

How to Run the Program:
1. Open the project folder in terminal or VS Code.
2. Run the file using: python main.py
3. Choose the desired option from 1-6 in the menu (choose 6 to exit)
